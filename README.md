## Ayaya
A fast and easy way to annoy your Discord friends.

### Commands
- `!ayaya`: Sends a predefined ayaya gif
- `!uiuiui`: Sends a predefined weird gif

### How to run
Set your discord token `export TOKEN=<your_token>`. Then
```
make && ./bin/ayaya
```
Annoy your friends :D
